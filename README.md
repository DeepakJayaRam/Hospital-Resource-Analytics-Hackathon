# Hospital Resource Utilization & Patient Outcomes Dashboard

## Problem Statement
Mid-sized Indian hospitals face bed shortages, ED congestion, high ALOS, and cost inefficiencies due to poor operational visibility.

## Objective
Design a data-driven dashboard that improves patient flow, resource utilization, and clinical outcomes.

## Solution Overview
- Defined 25+ hospital KPIs
- Designed 4 analytical dashboards
- Built star-schema-based data model
- Created DAX measures for hospital operations
- Generated actionable business insights

## Tools Used
- Power BI
- Excel / CSV
- GitHub

## Key Insights
- Orthopedics ALOS exceeds benchmark by 18%
- ED peak arrivals not aligned with staffing
- ICU bed blocking due to delayed discharges
